# Threads Auto Reveal

Auto-reveal hidden spoilers on Threads (text, images & videos).

## Install

[**Chrome Web Store**](https://chromewebstore.google.com/detail/ehcnpebbpcajfboepnlkmibfojccokdf)

## Features

- Automatically reveals hidden spoiler text, images, and videos on Threads
- Works instantly after installation — no configuration needed
- Toggle text and media reveal independently via the popup settings

## How It Works

The extension monitors the Threads page for spoiler-covered content. When detected, it automatically removes spoiler overlays so you can read and view everything without tapping "reveal" manually.

## Settings

Click the extension icon to open the popup, where you can toggle:

| Option | Description |
|--------|-------------|
| **Text** | Auto-reveal hidden spoiler text |
| **Media** | Auto-reveal hidden spoiler images & videos |

Both options are enabled by default.

## Privacy

- No data is collected or sent to any server
- Only runs on `threads.com`
- Uses `chrome.storage.sync` solely for storing your toggle preferences
